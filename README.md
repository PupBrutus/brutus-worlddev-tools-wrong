# Brutus WorldDev Tools

Editor utilities for VRChat world development.

## Tools
- Udon Behavior Profiler
- Lighting Analyzer
- Polygon Counter

## Menu Paths
- Tools/Brutus Scripts/Udon Behavior Profiler
- Tools/Brutus Scripts/Lighting Analyzer
- Tools/Brutus Scripts/Polygon Counter

## Notes
- These tools are editor-only and do not affect builds.
